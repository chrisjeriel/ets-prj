Read Me
  1. Create a folder and save the monthly DAT files from BIR Alpha system in that folder.
  2. Click Open File icon to select the folder where you saved the monthly DAT files.
  3. Click Import to import all the monthly details.
  4. Click Summary in tab Annual Details to consolidate per taxpayer.
  5. Click Export in tab Consolidated Data to create CSV data and DAT file in the folder.
